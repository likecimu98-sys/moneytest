1. Три дня на экране

Компромисс между таблицей и читаемостью: всегда виден вчера/сегодня/завтра, а перенос в соседний день становится естественным.

  • Лучший кандидат для основного мобильного режима.
  • Пустая ячейка нажимается для добавления.
  • Двойной тап по уроку может включать режим переноса.

2. Неделя сверху, день списком

Самый спокойный вариант: неделя сканируется по количеству уроков, выбранный день читается крупными карточками.

  • Самый безопасный по UX.
  • Не подходит, если важна таблица всех дней одновременно.
  • Хорош для ежедневного закрытия уроков.

3. Квадратная неделя

Полная неделя видна сразу. Внутри дня только мини-линии и точки, подробности раскрываются в нижнем блоке.

  • Хорошо для обзорности.
  • Меньше текста, меньше каши.
  • Подходит для темной темы.

4. Почти как ПК, но 5 дней скроллом

Это самый прямой ответ на идею табличной мобильной недели: видны 3-5 дней, остальное горизонтально.

  • Плюс: перенос через ячейки понятен.
  • Минус: часть недели скрыта за скроллом.
  • Нужно очень аккуратно настроить свайп, чтобы не менять неделю случайно.

5. Неделя по группам

Не основной календарь, а мощный режим для репетитора с группами: сразу видно регулярность и дырки по каждой группе.

  • Отлично для редактирования серий.
  • Можно быстро остановить или перенести группу.
  • Не подходит для индивидуальных учеников как основной экран.

6. Фокус-день + соседи

Выбранный день читается нормально, но соседние дни не исчезают. Хорошо для телефона и ежедневной работы.

  • Сильный вариант для главного режима.
  • Меньше перегруза, чем в полной таблице.
  • Соседние дни остаются кликабельными.

7. Неделя как чек-лист

Это не календарь, а режим работы: быстро закрыть прошедшие уроки, ДЗ, темы и долги.

  • Сильно повышает ежедневную полезность.
  • Подходит как отдельный режим “Закрытие”.
  • Не заменяет сетку расписания.

8. Расписание через свободные окна

Для коммерции важна не только занятость, но и где можно поставить нового ученика. Этот режим показывает окна явно.

  • Полезно при росте базы.
  • Пустое время становится действием.
  • Хорошо для “куда добавить урок”.

9. Неделя стопкой

Самый читаемый вариант без горизонтального скролла: все дни идут сверху вниз, выбранный день раскрывается.

  • Отлично для маленьких экранов.
  • Нет случайного свайпа недели.
  • Хуже ощущается как календарная сетка.

10. Отдельный режим переноса

Двойной тап по уроку включает режим переноса: выбранный урок подсвечен, свободные ячейки становятся явными целями.

  • Лучше, чем пытаться тащить пальцем по маленькой сетке.
  • Снижает риск случайного переноса.
  • Можно добавить подтверждение перед сохранением.
